## Proyectos destacados

### BASILE — SaaS multiempresa para carnicerías
Plataforma multi-tenant para la gestión integral del negocio: ventas, cuentas corrientes, apertura y cierre de caja, y roles de usuario.
**Stack:** FastAPI · SQLModel · React · TypeScript · PostgreSQL · Docker
[→ Ver repositorio](https://github.com/LfgLeonardoGomez/carniceria-management-system)

### FoodStore — Plataforma de pedidos online (Fullstack)
Sistema de pedidos y ventas online. Responsable del desarrollo completo del backend y del store de clientes. Incluye JWT en cookies HttpOnly, RBAC, patrón Unit of Work, WebSockets e integración con Mercado Pago.
**Stack:** FastAPI · PostgreSQL · React · TypeScript · Vite
[→ API](https://github.com/LfgLeonardoGomez/foodstore-api) · [→ Store cliente](https://github.com/LfgLeonardoGomez/foodstore-store-cliente) · [→ Admin](https://github.com/LfgLeonardoGomez/foodstore-admin)

### Gestión de proveedores con IA
Sistema de cuentas corrientes de proveedores con extracción automática de datos de facturas mediante IA (visión), para mantener los saldos siempre actualizados.
**Stack:** FastAPI · React · PostgreSQL · IA / visión
